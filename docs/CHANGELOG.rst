版本 0.1.0（当前版本）

**发布日期**: 2025-12-20

新增功能
--------
- ✅ 完整的用户认证系统
  - 用户注册功能
  - 用户登录/注销功能

  - 密码修改功能
- ✅ 基础投票功能

- ✅ 管理功能
  - Django 管理后台
  - 投票数据管理
  - 用户管理
- ✅ 完整测试覆盖
  - 10个测试全部通过
