Django Polls 文档
=================

欢迎访问 Django Polls 项目文档。

快速开始
--------

1. 安装依赖：
   .. code-block:: bash
      
      pip install django

2. 运行项目：
   .. code-block:: bash
      
      python manage.py runserver

3. 访问网站：
   打开浏览器访问 http://localhost:8000
