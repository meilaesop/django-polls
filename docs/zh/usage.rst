==========
使用指南
==========

本文档介绍如何使用 Django Polls 投票系统的各项功能。

🏠 快速开始
-----------

### 访问网站
- 网址：http://your-domain.com/polls/
- 或本地开发：http://127.0.0.1:8000/polls/

### 首次使用
1. **注册账号**：点击右上角"注册"按钮
2. **登录系统**：使用注册的账号登录
3. **开始使用**：浏览投票或创建新投票

👤 用户功能
-----------

### 用户注册
1. 点击页面右上角的"注册"按钮
2. 填写注册信息：
   - 用户名：用于登录的名称
   - 邮箱地址：用于接收通知（可选）
   - 密码：至少8位字符
   - 确认密码：再次输入密码
3. 点击"注册"按钮
4. 注册成功后会自动登录

### 用户登录
1. 点击页面右上角的"登录"按钮
2. 填写登录信息：
   - 用户名
   - 密码
3. 点击"登录"按钮

### 个人中心
登录后点击右上角用户名可以访问个人中心：

**功能包括：**
- 查看个人信息
- 查看投票记录
- 修改账号设置

### 退出登录
点击右上角用户名，选择"退出登录"

🗳️ 投票功能
------------

### 浏览投票
- **首页**：显示所有活跃的投票
- **查看详情**：点击投票标题进入详情页

### 参与投票
**步骤：**
1. 找到感兴趣的投票
2. 点击投票标题进入详情页
3. 选择一个选项（单选）
4. 点击"投票"按钮
5. 查看投票结果

**注意：**
- 每个用户对每个投票只能投一次
- 投票后不能修改

### 查看投票结果
投票后会自动跳转到结果页面：

**显示内容：**
- 各选项得票数
- 得票百分比
- 投票进度条
- 总投票人数

### 创建投票
**前提**：必须登录

**步骤：**
1. 点击"创建投票"按钮
2. 填写投票信息：
   - 投票问题：清晰描述投票主题
   - 选项1、选项2...：至少提供2个选项
3. 点击"创建"按钮

**权限：**
- 只有登录用户可以创建投票
- 可以编辑自己创建的投票

🏢 管理后台
------------

### 访问管理后台
- 网址：http://your-domain.com/admin/
- 或本地：http://127.0.0.1:8000/admin/

### 管理员功能
**用户管理：**
- 查看所有用户
- 禁用/启用用户账号

**投票管理：**
- 查看所有投票
- 审核投票内容
- 删除不当投票

### 成为管理员
```bash
# 在本地开发环境创建超级用户
python manage.py createsuperuser
```

❓ 常见问题

---

注册登录问题

Q：忘记密码怎么办？
A：目前需要联系管理员重置密码。

Q：可以修改用户名吗？
A：目前不支持修改用户名。

投票问题

Q：为什么我不能投票？
A：可能原因：

· 未登录（需要登录才能投票）
· 已经投过票（每人只能投一次）

Q：可以匿名投票吗？
A：目前不支持匿名投票，所有投票都需要登录。

Q：投票结果会公开吗？
A：是的，投票结果是公开的。

技术问题

Q：页面加载很慢？
A：尝试清理浏览器缓存或刷新页面。
